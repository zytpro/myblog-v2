# 博客项目开发文档

## 1. 项目概述

这是一个完整的个人博客项目，包含后端和前端两个部分。项目功能丰富，涵盖了文章管理、用户管理、多媒体资源管理、AI聊天室、留言板等多个模块，是一个综合性的个人博客系统。

### 1.1 项目结构

- **后端**：基于 Spring Boot 2.6.13 开发，采用 Maven 管理依赖
- **前端**：基于 Vue 3 开发，使用 Vite 作为构建工具

## 3. 后端架构与核心功能

### 3.1 架构设计

后端采用经典的三层架构：
- **控制器层（Controller）**：处理HTTP请求，调用服务层方法
- **服务层（Service）**：实现业务逻辑
- **数据访问层（Mapper）**：与数据库交互

### 3.2 核心功能模块

1. **用户认证模块**
   - 基于 Sa-Token 和 JWT 的认证授权
   - 登录、注册、权限管理
   - 支持记住登录状态

2. **文章管理模块**
   - 文章的增删改查
   - 文章分类和标签管理
   - 文章分页查询

3. **多媒体资源管理**
   - 图片、视频、音乐上传
   - 资源分类管理
   - 支持阿里云 OSS 存储

4. **AI 聊天模块**
   - 基于 OpenRouter API 的 AI 聊天功能
   - 支持多种 AI 模型

5. **留言板模块**
   - 留言的增删改查
   - 留言审核功能

6. **热搜模块**
   - 热搜API集成
   - 热搜数据管理

7. **轮播图管理**
   - 轮播图的增删改查
   - 轮播图状态管理

8. **导航栏管理**
   - 导航栏项的增删改查
   - 导航栏状态管理

9. **生活时光模块**
   - 生活记录的增删改查
   - 生活记录分类管理

10. **恋爱小屋模块**
    - 恋爱记录的增删改查
    - 恋爱时光管理

### 3.3 核心 API 接口

| API路径 | 方法 | 功能描述 |
|---------|------|---------|
| `/article` | GET | 获取文章列表（分页） |
| `/article/{id}` | GET | 获取文章详情 |
| `/article/save` | POST | 新增文章 |
| `/article/{id}` | PUT | 更新文章 |
| `/article/{id}` | DELETE | 删除文章 |
| `/article/tags` | GET | 获取文章标签 |
| `/article/categories` | GET | 获取文章分类 |
| `/navbar/list` | GET | 获取导航栏列表 |
| `/auth/login` | POST | 用户登录 |
| `/auth/register` | POST | 用户注册 |

## 5. 数据库结构与配置

### 5.1 数据库配置

- **数据库类型**：MySQL
- **连接配置**：
  - URL: `jdbc:mysql://localhost:3306/myblog`
  - 用户名: `root`
  - 密码: `root`

### 5.2 核心数据表

1. **users** - 用户表
   - id: 主键
   - username: 用户名
   - password: 密码（MD5加密）
   - avatar: 头像
   - email: 邮箱
   - rule: 角色（0-管理员，1-普通用户）
   - phone: 手机号
   - description: 描述

2. **articles** - 文章表
   - id: 主键
   - title: 标题
   - content: 内容
   - author: 作者
   - create_time: 创建时间
   - update_time: 更新时间
   - category: 分类
   - tags: 标签
   - status: 状态

3. **comments** - 评论表
   - id: 主键
   - article_id: 文章ID
   - user_id: 用户ID
   - content: 评论内容
   - create_time: 创建时间
   - status: 状态

4. **categories** - 分类表
   - id: 主键
   - name: 分类名称
   - description: 描述

5. **tags** - 标签表
   - id: 主键
   - name: 标签名称

6. **carousel** - 轮播图表
   - id: 主键
   - image_url: 图片URL
   - link_url: 链接URL
   - title: 标题
   - status: 状态

7. **navbar** - 导航栏表
   - id: 主键
   - name: 导航名称
   - path: 路径
   - icon: 图标
   - status: 状态

8. **message** - 留言表
   - id: 主键
   - user_id: 用户ID
   - content: 留言内容
   - create_time: 创建时间
   - status: 状态

9. **life_time** - 生活时光表
   - id: 主键
   - title: 标题
   - content: 内容
   - image_url: 图片URL
   - create_time: 创建时间

10. **love_room** - 恋爱小屋表
    - id: 主键

## 6. 项目特点与优势

1. **功能丰富**：涵盖了博客系统的核心功能，包括文章管理、用户管理、多媒体资源管理、AI聊天室、留言板等
2. **技术栈现代**：后端使用 Spring Boot 2.6.13 + Java 17，前端使用 Vue 3 + Vite，技术栈先进
3. **架构清晰**：后端采用三层架构，前端采用组件化开发，代码结构清晰
4. **安全性高**：使用 Sa-Token 和 JWT 进行认证授权，密码使用 MD5 加密
5. **响应式设计**：前端使用 Element Plus 和 Vuetify 实现响应式布局，适配不同设备
6. **性能优化**：使用 Redis 缓存，PageHelper 分页，提高系统性能
7. **扩展性强**：模块化设计，易于添加新功能
8. **用户体验好**：界面美观，交互流畅，功能完善

## 8. 总结

本项目是一个功能完整、技术栈现代的个人博客系统，具有以下特点：

1. **技术栈先进**：后端使用 Spring Boot 2.6.13 + Java 17，前端使用 Vue 3 + Vite，技术栈符合现代开发标准。
2. **功能丰富**：涵盖了文章管理、用户管理、多媒体资源管理、AI聊天室、留言板等多个模块，功能完善。
3. **架构清晰**：后端采用三层架构，前端采用组件化开发，代码结构清晰，易于维护和扩展。
4. **用户体验好**：界面美观，交互流畅，响应式设计适配不同设备。
5. **安全性高**：使用 Sa-Token 和 JWT 进行认证授权，密码加密存储。

通过本项目的开发，可以学习到现代前后端分离架构的开发流程，掌握 Spring Boot、Vue 3 等技术栈的使用，以及如何构建一个完整的博客系统。

同时，项目还有一些可以改进的地方，如安全性提升、性能优化、功能完善等，这些改进可以使项目更加成熟和稳定。
